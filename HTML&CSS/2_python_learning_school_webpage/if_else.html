If & Else Statements


Syntax : if:

Syntax : elif:

Syntax : else:

Input:

        name = "gaurav singh"

        if name == "vishwa":
            print("Name Is Incorrect")

        elif name == "Deep":
            print("Name Is wrong")

        else:
            print("Name Is gaurav singh")

OutPut:

Name Is gaurav singh

Dictionary


Syntax : {} or dict()

Input:

        result = {"Rajesh":45, "Vijay":55, "Mohit":65, "Ajay":87, Pankaj:42]

        print("Students Names & Marks Are:"result)

OutPut:

Students Names & Marks Are: Rajesh: 45, Vijay: 55, Mohit: 65, Ajay: 87, Pankaj: 42
